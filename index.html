Google Theme

This is google theme of select style. You just need to write google in theme option. This is also basic theme

Dark Theme

This is dark theme of select style. You just need to write dark in theme option.

Documentation

The following options are available for select tag:

		theme = google
		Theme for different color

		width = 250
		Width of selector

		placeholder = null
		Placeholder of selector for first time.

		data-search = true
		Search bar of selector.

The following options are available for options:

		font-family : null
		Font Family of selected option

		data-image : null
		Image of selected option.(max width and height is 16px)

Let’s take a look at our custom select script. Having a select element like the following

	<select id="myselectid" theme="google" width="250" style="" placeholder="Select Your Favorite">
		<option>Themeforest</option>
		<option>Codecanyon</option>
		<option>Videohive</option>
	</select>

we’ll transform it into this structure:

	<div class="selectstyle ss_dib google" style="width:260px;">
		<div id="select_style" class="ssbutton" style="width:260px;">
			<div class="ss_dib" id="select_style_text" style="width:240px;">Select Your Favorite</div>
			<div class="ss_dib ss_image"></div>
		</div>
		<ul id="select_style_ul" sid="myselectid" style="" class="ss_ul">
			<li data-title="Themeforest" value="Themeforest">Themeforest<li>
			<li data-title="Codecanyon" value="Codecanyon">Codecanyon<li>
			<li data-title="Videohive" value="Videohive">Videohive<li>
		</ul>
	</div>
